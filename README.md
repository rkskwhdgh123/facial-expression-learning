# facial-expression-learning

---

기쁨, 슬픔, 놀란, 화난 4가지의 표정을 학습합니다.  


카메라로 사람의 얼굴을 찾고 인식되면 어떤 표정을 짓는지에 따라 그위에 표정 이미지를 씌워 줍니다. 

```

예를들어 슬픈 표정을 하고 있을때는 이렇게 슬픈 표정의 이미지를 씌워줍니다.
```
